PC-relative - Difference between address of target instruction and address of the instruction immediately following jump - This offset can be encoded using one, two or four bytes

cmovC src, dest - Move value from src to dest if condition C holds More efficient than conditional branching (simple control flow) - If evaluation requires a significant computation it is wasted - GCC uses conditional move when the two expressions can be computed very easily But overhead: both branches are evaluated
